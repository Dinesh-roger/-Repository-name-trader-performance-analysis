# Trader Performance vs Market Sentiment

## Methodology
- Cleaned trader and sentiment datasets
- Merged on daily timestamps
- Created metrics: PnL, win rate, leverage, trade frequency
- Compared Fear vs Greed behavior

## Key Insights
1. Traders perform better during Greed phases with higher average PnL.
2. High leverage increases risk without consistent profitability.
3. Frequent trading does not guarantee higher returns.

## Strategy Recommendations
- Reduce leverage during Fear markets.
- Focus on controlled position sizing.
- Avoid overtrading in volatile sentiment phases.
